# RunQuickSimulator

quick3.5不再使用player, 换成Simulator. 启动的方式有三种:  

+ cocos run
+ debug_win.bat(windows)
+ debug_mac.sh (mac)

都很麻烦.  
做了一个可以在sublime直接启动simulator的插件.  


## 启动方式
+ 打开项目文件, 在src下的lua文件.
+ 快捷键: F1
